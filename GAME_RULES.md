# Frontwave ゲームルール仕様書

## 概要

Frontwave はリアルタイム戦略ゲームです。プレイヤーは6つの国家の1つを操作し、州（province）を支配・拡大して勝利を目指します。

---

## 国家

| ID    | 名称   | 特性         | 概要                     |
|-------|--------|--------------|--------------------------|
| `emp` | 帝国   | defense      | 防衛特化。高い守備力と安定した開発力 |
| `kgd` | 王国   | attack       | 攻撃特化。積極的な侵攻と高いDPS |
| `fed` | 連邦   | mobility     | 機動特化。海峡越えと素早い展開 |
| `rep` | 共和国 | economy      | 経済特化。高い収入と投資効率 |
| `hol` | 聖王国 | terrain      | 地形防衛特化。山岳・首都防御が強力 |
| `dch` | 公国   | ambush       | 奇襲特化。妨害・破壊工作に優れる |

---

## ゲームモード

### カジュアルモード (`casual`)
- デフォルトタイマー：**1200秒（20分）**
- 時間切れ時、**最多州数**の国家が勝利（同数なら引き分け）
- 全土制圧でも即時勝利

### ストラテジーモード (`strategy`)
- タイマーなし
- **全土制圧**（最後の生存国家になる）のみ勝利条件
- 首都陥落で国家が即時降伏し、全州が攻略国に移管される

## 戦闘

### 基本ルール

- 攻撃は**隣接州**または**海峡越え**（条件あり）にのみ可能
- **宣戦布告後**でないと攻撃不可
- 攻撃元の兵力が **20未満** の場合は攻撃不可
- **交戦中の州は兵力再生しない**

### ランチェスターの法則

毎ティック（0.1秒）ごとに以下の式でダメージが計算される：

```
攻撃側ダメージ = 攻撃側兵力 × 0.6 × 攻撃側補正 × Δt
防衛側ダメージ = 防衛側兵力 × 0.8 × 地形防衛倍率 × 要塞化倍率 × Δt
```

### 攻撃側補正

| 条件                     | 補正値 |
|--------------------------|--------|
| 通常攻撃                 | ×0.75  |
| 電撃戦カード適用時       | ×1.0（ペナルティなし） |
| 同盟破棄ペナルティ中     | ×0.7   |
| 攻撃ブーストカード       | ×倍率  |

### 戦闘解決

- **攻撃側の兵力が0以下** → 攻撃失敗、防衛側の州はそのまま
- **防衛側の兵力が0以下** → 攻略成功
  - 攻略後の攻撃側兵力：`(残存兵力) × 0.8`
  - 占領した州の初期兵力：`max(1, 攻撃側残存兵力 × 0.5)`
  - 新たな占領で相手と外交関係が戦争状態でない場合は自動宣戦に昇格

### 首都陥落（ストラテジーモード）

首都が占領されると、その国家は即時**降伏**し全州が攻略国に移管される。

## 外交

### 関係ステータス

| ステータス | 意味                     |
|------------|--------------------------|
| peaceful   | 平和。攻撃不可            |
| war        | 戦争中。相互攻撃可        |
| ally       | 同盟。兵力移送可、攻撃不可 |

### 同盟

- 1国家あたり最大 **2同盟** まで
- プレイヤーが同盟提案 → 相手AIが条件を判断して承認/拒否（TTL: 20秒）
- 同盟の条件（AI）：提案国の兵力が自国の80%以上かつ共通の敵が1つ以上

### 同盟破棄ペナルティ

同盟を破棄して宣戦した場合：
- 攻撃力 **-30%（×0.7）** が **180秒間**継続

---

## 兵力移送

- 自国の隣接州間のみ移送可能
- 同盟国の隣接州にも移送可能
- 移送量は `min(指定量, 保有兵力 - 1)` まで（最低1兵力を残す）
- 移送先の上限を超える分は切り捨て

## カードシステム

### ドラフト

- **60秒ごと**にカードドラフトが発生
- 手札が上限（5枚）に達している場合はスキップ
- ドラフト時に3枚提示され、プレイヤーが1枚選択（またはスキップ）
- 手札は**プレイヤーの国家専用**カードプール（12枚）からドロー

### カード発動

- 手札からカードを選んで発動
- `needsTarget: true` のカードは対象州の指定が必要
- 発動後はすぐ手札から消える

## 勝利条件

### カジュアルモード
1. **全土制圧**：他の全国家が消滅（即時勝利）
2. **タイムアップ**：最多州数の国家が勝利。同数なら引き分け

### ストラテジーモード
1. **全土制圧**：他の全国家が消滅

## AIの行動

### 攻撃判断
- 2秒ごとに行動（AI_TICK_INTERVAL）
- 同時攻撃数の上限：2件
- 国家特性によって攻撃閾値（自軍/敵軍比率）が異なる

| 特性     | 攻撃閾値 |
|---------|---------|
| defense | 1.5     |
| attack  | 1.1     |
| mobility| 1.2     |
| economy | 1.4（序盤120秒は2.0） |
| terrain | 1.5     |
| ambush  | 1.0     |

### 宣戦布告
- 8秒ごとに宣戦を評価（AI_WAR_DECISION_INTERVAL）
- スコアが0以上の相手に宣戦
- 同盟相手への宣戦は-50ペナルティ、既存の戦争1件ごとに-25

### 開発投資
- 30秒ごとに最低産業レベルの州を自動アップグレード開始
- 国家特性によって上限レベルが異なる（defense: 5、attack: 3 など）

### 同盟応答
- 「自国兵力≥80%かつ共通の敵が1人以上」の場合に承認

## 海峡越え

- 通常は海峡を挟んだ州への攻撃は不可
- 以下の条件で海峡越えが可能：
  - 海峡越えカード（`strait_crossing_enabled`エフェクト）を使用中
- 海峡のルートはマップデータで定義
