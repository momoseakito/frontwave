# EUROPA BELLUM — ゲーム設計書 v1.0（仮案）

---

## 1. ゲームコンセプト

架空のヨーロッパを舞台にした近世風リアルタイムストラテジー。6つの架空国家が州の覇権を争う。スマホ・PC両対応のブラウザゲームとして設計し、広告によるマネタイズを採用する。

| 項目           | 内容                                 |
| -------------- | ------------------------------------ |
| タイトル（仮） | EUROPA BELLUM                        |
| 舞台           | 架空ヨーロッパ・近世風（15〜17世紀） |
| プレイスタイル | リアルタイム同期型                   |
| 対応プレイ     | ソロ（vs AI）＆ マルチプレイヤー     |
| 対応端末       | スマホ（縦画面）・PC 両対応          |
| マネタイズ     | 広告                                 |

## 5. コアゲームループ

**州占領 → ゴールド・兵力生産UP → さらに攻勢 → 国家方針カードで強化 → 同盟で外交展開**

| フェーズ | 内容                                                       |
| -------- | ---------------------------------------------------------- |
| 序盤     | 中立州を素早く取り合い、初期兵力を確保                     |
| 中盤     | 州開発でゴールド・生産力を伸ばしながら国家方針カードを蓄積 |
| 後半     | 同盟を活用しつつ敵首都を狙う大規模攻勢                     |

## 6. 兵力生産システム

州ごとに毎秒自動で兵力が生産される。開発レベルが上がるほど生産速度・上限が増加する。

| 州の種類      | 毎秒生産 | 上限兵力 |
| ------------- | -------- | -------- |
| 未開発（Lv0） | 1 / 秒   | 1,000    |
| 開発 Lv1      | 2 / 秒   | 1,800    |
| 開発 Lv2      | 4 / 秒   | 2,800    |
| 開発 Lv3      | 7 / 秒   | 4,000    |
| 開発 Lv4      | 11 / 秒  | 5,500    |
| 開発 Lv5      | 16 / 秒  | 7,500    |

### 首都州の特別ルール

| 項目                   | 内容                           |
| ---------------------- | ------------------------------ |
| 生産量                 | 開発Lvに関わらず +50% ボーナス |
| 上限兵力               | 開発Lvに関わらず +50% ボーナス |
| 占領難度               | 通常の2倍の兵力が必要          |
| 陥落時（ストラテジー） | その国は降伏                   |

### 中立州

| 項目     | 内容             |
| -------- | ---------------- |
| 初期兵力 | 10               |
| 毎秒生産 | 1 / 秒           |
| 上限兵力 | 200              |
| 開発     | 占領後に開発可能 |

## 9. 戦闘システム

ランチェスターの法則をベースにした毎秒ダメージ計算。防衛側が若干有利な設計。

```
毎秒攻撃側が受けるダメージ = 防衛側兵力 × 防衛係数
毎秒防衛側が受けるダメージ = 攻撃側兵力 × 攻撃係数
```

| 係数     | 値                       |
| -------- | ------------------------ |
| 攻撃係数 | 0.1（仮）                |
| 防衛係数 | 要調整（1.1〜1.5の想定） |

> ※ 防衛係数は実装後のゲームバランス調整で確定する。

## 11. 同盟システム

| 項目       | 内容                     |
| ---------- | ------------------------ |
| 締結コスト | 無料                     |
| 同盟上限   | 最大2カ国                |
| 締結方法   | 相手が承認すれば即時成立 |

### 同盟中の機能

| 機能     | 内容                         |
| -------- | ---------------------------- |
| 不可侵   | 同盟中は互いに攻撃不可       |
| 情報共有 | 同盟国の全州の兵力数が見える |
| 共同宣戦 | 第三国に一緒に宣戦できる     |

### 同盟破棄ペナルティ

| 項目           | 内容             |
| -------------- | ---------------- |
| ペナルティ内容 | 全軍の攻撃力-30% |
| ペナルティ時間 | 3分間            |
| 適用対象       | 破棄した側のみ   |

> ※ 援軍送付機能は将来的な追加機能として検討。
